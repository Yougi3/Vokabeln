Signaler une erreur de traduction

Il est possible que tu es rencontré une erreur dans la traduction d'un mot.
Pour que je puisse corriger l'erreur et offrir une meilleure expérience de révision à tout le monde, suis les étapes ci-dessous.

PS : Je prévois de simplifier le processus dans le futur.

1. Crées un compte GitHub

GitHub est une plateforme que les développeurs utilisent pour partager leur code. Ne t'inquiètes pas, c'est sans danger !
Crées un compte en te rendant sur la page d'accueil de GitHub.
Renseigne un nom d'utilisateur, une adresse mail et un mot de passe.


2. Accèdes à la page GitHub du projet

Rends-toi à l'adresse suivante : https://github.com/Yougi3/Vokabeln/. C'est l'adresse de la page GitHub du projet. Tu peux alors voir le code qui fait fonctionner le site sur lequel tu te trouves actuellement. (C'est trop cool n'est-ce pas ?)


3. Cliques sur 'Issues' puis 'New issue'

Une issue c'est un rapport d'erreur ou une suggestion. C'est grâce à ces rapports que les développeurs sont avertis de la présence d'un problème dans leur code.


4. Décris le problème

Dans la description, précise de quel mot il s'agit avec son orthographe actuel et l'orthographe corrigé.


5. Cliques sur 'Submit new issue'

Et voilà ! Tout est terminé. Je pourrais ainsi investiguer le problème et le corriger au plus vite.
